# PlusMin Redzaamheid Theme

Dit WordPress theme is opnieuw opgebouwd voor PlusMin.

## Kenmerken

- Volledig nieuw basisontwerp afgestemd op de missie van PlusMin.
- PlusMin logo in header en footer.
- Lorem Ipsum voorbeeldteksten.
- Kleur-placeholders in plaats van echte afbeeldingen.
- Responsive layout voor desktop en mobiel.
